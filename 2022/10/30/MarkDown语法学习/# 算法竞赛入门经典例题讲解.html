算法竞赛入门经典例题讲解

回文词（Palindromes,UVa401）

本文会对该题进行分析和分享自己的见解。

问题描述

输入一个字符串，判断它是否为回文串以及镜像串。输入字符串保证不包含数字0.所谓回文串，就是反转后和原串相同。如abba,abccba,madam。而镜像串，就是左右镜像后原串相同。把一个字符串的一半字符，镜像翻转后和原来剩下一半的字符相同。如2s（2镜像翻转后变成s）、3AIAE（3镜像翻转为E，A镜像翻转为A）

思路分析

  • 回文串的判断：要想一个字符串是回文串，即从正向读和反向读相同，只要第i个字符和倒数第i个字符相同即可。
  • 镜像串的判断：
    镜像表（并非所有字符都有镜像，没有镜像的有空格表示）
    字符 镜像 字符 镜像 字符 镜像
    A A M M Y Y
    B N Z 5
    C O O 1 1
    D P 2 S
    E 3 Q 3 E
    F R 4
    G S 2 5 Z
    H H T T 6
    I I U U 7
    J L V V 8 8
    K W W 9
    L J X X
    我们只要把上面的表格按顺序用数组制成表格，如数组索引0代表字符A的镜像。。。
    我们根据字符的运算很容易就能算出相应的索引了。但我们制作的表格大写字母和数字是连续的
    而ASCALL码里却是不连续的，所以我们在计算字母的镜像索引和计算数字的镜像索引要分开计算。

代码实现

{
    #include<stdio.h>
    #include<string.h>
    #include<ctype.h>
    const char* rev ="A   3  HIL JM O   2TUVWXY51SE Z  8 " //制作镜像表
    const char* msg[] = {"not a palindrome","a regular palindrome","a mirrored string","a mirrored palindrome"}//四种组合，目的是为了减少消耗，简便化，下面会进一步说明。
    
    //镜像串的判断函数
    char r(char ch){
        if(isalpha(ch)) return rev[ch-'A'];//isalpha函数是用来判断是否为字母，声明包含在ctype.h头文件
        return rev[ch-'0'+25];
    }
    
    int main(){
        char s[30];
        while(scanf(%s,s)==1){
            int len = strlen(s);
            int p = 1,m=1;//用于回文串和镜像串状态判断，p有1，0态，代表是否回文串；m有1，0态，代表是否镜像串.
            for(int i;i<(len+1)/2;i++){
                if(s[i]!=s[len-1-i]) p=0; //不是回文串
                if(r(s[i])!=s[len-1-i]) m=0;//不是镜像串
            }
            printf("%s -- is %ss.\n\n",s,msg[m*2+p]);
            //msg[]就是用于这里的效率
            //msg有四种状态，不是回文串非镜像，是回文串非镜像，是镜像串非回文，是镜像回文串。
            //为什么回文状态p*1，而镜像状态m**2？
            //分析msg[0],msg[1],区别是否回文，下标差为1；msg[2],msg[3]区别为是			否回文，下标差为1.而msg[0]和msg[2]区别为是否镜像，下标差为2，msg[1]和msg[3]区别是是否镜像,下标差为2。可以简单理解为权重。我们也可以修改msg状态顺序来让实现msg[p*2+m]，只要改成msg[] = {"not a mirrored string","a mirrored sring"," a palindrome","a mirrored palindrome"}
        }
    }

}
