定义函数

函数参数的传递和其他语言一样，都是传值。

传递参数

参数可以是数字，字符串，列表，元组，字典，函数参数是形参，对于非列表，元组，字典的修改，在函数外是不生效的
python函数参数的传递有两种方式，位置参数和指定参数

  1. 位置参数
    python根据位置来传递
  2. 指定参数
  3. 默认值
    在定义函数时可以设定默认值，如果函数需要的参数包含默认值和非默认值，则默认值要放在非默认值后面 设定好默认值后，缺少的参数会按照默认值传递，若不缺少则按照传递的值来穿传递,位置传参，指定传参，默认值可以混合使用
  4. 可选参数 上面pet_age变成了可选参数，如果输入有传递pet_age的参数就输出，没有则不输出

返回值

自己定义的函数可以没有返回值如定义一个打印函数，排序函数等，如果想要在调用完函数后返回一个值，就在函数的末尾加上return 返回值。返回值可以是任意数据类型
整型，字符串，列表，字典。

在函数中修改列表

传列表进函数默认是传原件进，在函数内修改列表，也会影响列表在函数外的值。

禁止函数修改列表

如果我们不想让函数修改列表原有的值。我们可以传列表的副本进函数，这样就避免了函数修改原列表。

传递任意数量的参数

如果我们事先不知道要传递多少个参数给函数，我们可以def printname(* name)的语句，python会把我们传递的参数全部封包到name这个元组里，无论是一个还是多个。

把函数储存在模块里

函数可以把代码和主程序分开来，通过给函数描述性名称，使得主程序代码更加容易理解，但如果函数过多，和主程序都编写在一个文件里，就显得臃肿了。我们可以把函数写在模块里，再通过import语句调用模块。

例如
我们先创建一个pizza.py文件，里面除了make_pizza函数，其他代码都不写

我们再在pizza.py文件的同目录下创建一个making_pizzas.py文件,这个文件导入pizza.py文件，然后再调用函数

导入特定的函数

如果我们创建了一个包含很多函数在funtion.py，但我们只需其中的一个函数，我们可以使用下面的语法
from fution import funtion_name
导入多个特定的函数
from fution import funtion_name1,funtion_name2
而且如果使用这个方式导入函数，可以直接使用函数名来调用函数
