python列表

列表可以储存多个数据，相等于c语言的数组。

列表的创建

创建列表的第一种方式使用[]

list1 = [“Hello”,”World”,98,87]


创建列表的第二种方式 使用内置函数list()

list2 =list([“Hello”,”World”,98,87])

列表的特点

  1. 有序
  2. 可以储存重复数据
  3. 可以储存不同的类型
  4. 动态分配内存和回收空间
  5. 索引对应唯一的值

列表的索引

index()

lst =[“Hello”,”World”,98,”Hello”]

print(lst.index(“Hello”))

输出结果 0

如果列表有多个相同元素，索引为相同元素里的第一个，如果列表不含有该元素，会报错，

lst =[“Hello”,”World”,98,”Hello”]

print(lst.index(“Hello”，1，3))

输出结果 会报错说不存在该元素
因为我们指定它从1-3元素里查找

获取列表特定的元素

lis[“Hello”,”World”,98]

正向索引是从0到（N-1）
逆向索引是从（-N）到(-1)

如果我们想获得列表中的单个元素就用索引，想获得多个元素就用切片

列表元素的增、删、改

列表元素的增加
1. 向列表末尾添加一个元素
lis = [10,20,30]
print(lis,id(lis))
lis.append(100)
print(lis,id(lis))
//添加前后都是同一个列表对象
2. 向列表末尾添加添加至少一个元素
lis = [10,20,30]
print(lis,id(lis))
lis2 = [“Hello”,”World”]
lis.extend(lis2)
print(lis,id(lis))
3.向列表任意位置插入元素
lis = [10,20,30]
print(lis,id(lis))
lis.insert(1,100)
print(lis,id(lis))
4.向列表任意位置切片至少一个元素
lis = [10,20,30]
print(lis,id(lis))
lis3 = [“Hello”,”World”]
lis[1:] = lis3
print(lis,id(lis))
—–运行结果 [10,”Hello”,”World”]
列表元素的删除
1. remove()
remove 一次删除一个元素，有相同元素时只删除第一个，若元素不存在报错
2. pop()
根据索引删除指定元素，如果不写索引，默认最后一个
3.切片
一次最少删除一个元素，切片后是新的列表对象
lis = [“Hello”,”World”,98,35]
new_list = lis[1:2]
不产生新列表对象，用空列表覆盖
lis = [“Hello”,”World”,98,35]

    lis[1:2] = []
4. clear()
   清除列表所有元素
   lis =["Hello","World",98,35]
   lis.clear()
 5. del
    删除列表对象
    del lis
    删除列表中的元素
    del lis[2]

列表的修改

  1. 指定索引元素修改
    lis = [10,20,30,40,50,60]
    lis[2] = 0
  2. 指定切片元素修改
    lis =[10,20,30,4,0,50,60]
    lis[1:3] = [100,200,300,400]

列表的排序

  1. 调用sort()函数，列表中元素默认按从小到大排序，reveser=True,进行降序
    list = [10,30,50,4,50,10,100,9]
    list.sort()
    list.sort(reverser =True)
  2. 调用内置函数sorted(),可以指定reverse=True 进行降序排序，原列表不发生改变
    lis = [10,30,50,40,10,100,9]
    new_list = sorted(list)
    decl_list = sorted(lis,reverser =True)
