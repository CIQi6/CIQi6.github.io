类

基于类创建对象时， 每个对象都自动具备这种通用行为，然后可根据需要赋予每个对象独 特的个性。

创建和使用类

下面是模拟小狗的一个类
class Dog():
“””一次模拟小狗的简单尝试”””

在python规范中首字母大写的名称为类。
类中的函数叫做方法。
方法__init__是一个特殊的方法，每个类都要有，当你根据Dog类创建实例时，python都会自动运行它，__init__包含三个参数self name age，其中self是必不可少的，且在最前面，因为python在调用__init__时将自动传入参数self，它是一个指向实例本身的引用，让实例能够访问类中的属性和方法。以self为前缀的变量可供类中使用方法使用，self.name = name获取存储在形参name中的值，并将 其存储到变量name中，然后该变量被关联到当前创建的实例。可以通过实例访问的变量叫属性。(注意init前后面分别是两个下划线)

初始化属性

类中的每个属性都要有初始值，哪怕是0或空字符串。设定初始值后，你可以不提供该属性的参数，来创建一个实例。

继承(inheritance)

当我们想要创建拥有另一个类里面属性的类，我们不必从零开始。我们可以通过继承的方法来创建，原类我们称它为父类，新创建的类我们叫子类。需要注意的是，创建子类，父类要在同一文件，且父类在子类前面。（我知道你爸爸，就能大概知道你的样貌，儿子和爸爸有相同，也有独特）

  1. 子类的方法__init__()
  2. 给子类定义新属性

重写父类

如果父类的属性在子类是不合适的，我们可以在子类中，重新定义这个属性来覆盖父类中的属性。

将实例用作属性

当我们对一个类添加非常多的属性时，整个类就显得十分臃肿。我们可以把这些属性归类，把某一类的属性定义在一个类里，再供我们使用。如我们上面写的电动车，我们把有关电动车电池的一些属性放在一起，如电池容量，显示电池容(descriptive_battery),剩余电量能行走的里程放在class Battery()中。我们接着在class ElectricCar() 中self.battery = Battery()，写这句语法，我们就能把Battery储存在ElectriCar了。
my_tesla.battery.descriptive_battery()，这样我们也可以调用查看电池容量。

导入类

和函数一样，我们也可以把类全部编写在一个文件里，供另一个文件使用。导入方法和导入函数的一样。
