创建数组的方法

以上三种方法结果是一样的，注意一下输入结果是数组

array：将输入数据（可以是列表、元组、数组以及其它序列）转换为ndarray(Numpy数组)，如不显示指明数据类型，将自动推断，默认复制所有输入数据。

arange：Python内建函数range的数组版，返回一个数组。

array的属性：

  • shape：返回一个元组，表示 array的维度 [形状，几行几列] （2，3）两行三列，（2，2，3）两个两行三列
  • ndim：返回一个数字，表示array的维度的数目
  • size：返回一个数字，表示array中所有数据元素的数目
  • dtype：返回array中元素的数据类型

使用ones创建全是1的数组

参数：

  • shape：整数或者整型元组定义返回数组的形状；可以是一个数（创建一维向量），也可以是一个元组（创建多维向量）
  • dtype : 数据类型，可选定义返回数组的类型。
  • order : {‘C’, ‘F’}, 可选规定返回数组元素在内存的存储顺序：C（C语言）-rowmajor；F（Fortran）column-major。

ones_like 创建形状相同的数组

参数：

  • a：用a的形状和数据类型，来定义返回数组的属性
  • dtype ： 数据类型，可选
  • order顺序 ： {‘C’，’F’，’A’或’K’}，可选,覆盖结果的内存布局。
  • subok ： bool，可选。True：使用a的内部数据类型，False：使用a数组的数据类型，默认为True

zeros 创建全是0的数组

参数：

  • shape：整数或者整型元组定义返回数组的形状；可以是一个数（创建一维向量），也可以是一个元组（创建多维向量）
  • dtype : 数据类型，可选定义返回数组的类型。
  • order : {‘C’, ‘F’}, 可选规定返回数组元素在内存的存储顺序：C（C语言）-rowmajor；F（Fortran）column-major。

zeros_like创建形状相同的数组

用法与ones_like 一样，详细参考[[#ones_like 创建形状相同的数组|ones_like]]

full创建指定值的数组

参数：

  • shape：整数或者整型元组定义返回数组的形状；可以是一个数（创建一维向量），也可以是一个元组（创建多维向量）
  • fill_value：标量（就是纯数值变量）
  • dtype : 数据类型，可选定义返回数组的类型。
  • order : {‘C’, ‘F’}, 可选规定返回数组元素在内存的存储顺序：C（C语言）-rowmajor；F（Fortran）column-major。

多维数组

numpy不仅可以处理一维，二维(我们通常也叫矩阵)，还可以处理n维，方法类同。

数组的维度转换

我们如果想要把一个一维数组变成二维数组，或者多维，又或者想把一个多维数组变成一维数组。我们需要用reshape()。在使用reshape时要注意保证前后两个形状的元素数目要相同，例如一个一维数组有一行十列共10个元素，我们可以把他转变成一个二维数组有两行五列，但不能转化成一个两行四列，元素数目不为10的数组。

flatten()：作用是把一个不知道元素数目的数组展开成一个一维数组

注意区分:
shape() ：返回数组的形状
reshape() ：不改变数值，改变形状.

数组和常数的四则运算

加法

一个数组A和一个常数C相加，相当于数组的每个元素(a1,a2,….,an)都加上这个常数C。可以理解为数组和常数相加时，先把常数变成一个和数组A形状相同的，且每个元素都为常数C的数组，再把两个数组相加。
