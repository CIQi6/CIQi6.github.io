数据的表现形式和运算

  1. 常量
    1. 整型常量：1，2，621，5，15111
    2. 实型常量：1.24，3.14159
    3. 字符常量
      1. 普通字符：’a’,’A’,’n’,’1’
      2. 转义字符：’\n’,”\t”
    4. 字符串常量：“hello”
    5. 符号常量：#define pi 3.14159
  2. 变量 :先定义再使用
    (数据类型) 变量名 ; //定义一个变量
    当我们定义了一个变量就是编译器按数据类型所需的内存大小给我们找了一块连续的内存，让我们可以在那里储存数据，并按照我们设定的变量名与那块内存的地址关联起来（在程序编译连接时）。从变量中取值实际是通过变量名得到内存地址，在从内存中获取数据。
  3. 常变量
    在定义变量时在前面加上const
    如 const int a=5;
    定义后常变量和常量一样不能被改变了
  4. 标识符
    标识符简单讲加上名字

数据类型

  1. 基本类型
    1. 整型
      整型有有符号和无符号两种，储存整型的内存中会有一位用来表示符号
      1. 整型int
        一般为2或4字节，unsigned int 范围
      2. 短整型short
        2字节
      3. 长整型long int
        4字节
      4. 长长整型long long int
        八字节
      5. 字符型 char
        字符在内存中是以整型形式存储的。有专门的标对照，如anscall码。。
        1字节
      6. 布尔类型 bool
        1字节
    2. 浮点型：
      储存浮点型数据内存有符号位、小数部分、指数位组成
      1. 单精度类型 float
        能显示6位有效数字
      2. 双精度类型 double
        能显示15位有效数字
  2. 空类型viod
  3. 枚举类型enum
  4. 派生类型
    1. 指针类型 *
    2. 数组类型 arr[]
    3. 函数类型
    4. 结构体类型 struct
    5. 共同体类型 union

}

c语言中可以函数内和函数外定义变量。

全局变量和局部变量

  1. 在函数外定义的变量叫全局变量，它是静态的，在每一个函数都可以调用。
  2. 在函数内部定义的变量叫局部变量，只在定义变量的函数起作用，在别的函数里面不起作用。 定义局部变量，编译器不会帮你初始化变量，而定义全局变量时会帮你初始化。
    /tab

static 静态变量

[[static修饰]]

用static修饰变量用，就可以让变量在函数调用完后不被释放。下次再次调用这个函数变量的值是上次调用结束的值

按位运算

  • & 按位与
    如果$x_i$与$y_i$都为1，则为1，如
    1010 1010
    x 1100 1010
    = 1000 1010
    应用：1.使某一位或几位为0
    2.获取一个数的某一段
    如：1111 1110
    x 1010 1011
    1010 1010 这样就得到了1010 1011的7位

  • | 按位或
    如果$x_i$与$y_i$有一个为1，则为1，如
    1010 1010
    x 1100 1010
    = 1110 1010
    应用：1.使某一位或几位为1
    2.拼接两个二进制数

  • ~ 按位取反 （~x）= 1-x
    1变成0，零变成1

  • ^ 按位异或
    两个为相等则为0，两个不相等则为1.
    对一个数做一次^，再对得到的结果做一次^就会得到原来的数

移位运算

<< 左移
i << j 将i中的所有位向左移j个位置，右边用0填充
i <<= 1 等价于 i * =2 why? —-> 1 1 1 1 ($2^3 +2^2 +2^1 +2^0$)假设向左移动一位 1 1 1 1 0($2^4 +2^3 +2^2 +2^1$)
i << = n 等价于 i * = $2^n$
.>> 右移
i << j 将i中的所有位向右移j个位置
对于int型，右移后不变
对于unsigned 右移
对于signed 符号不变
对于无符号数，右移之后高位补0；对于有符号数，符号位一起移动，正数高位补0，负数高位补1
